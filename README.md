# DSUtil

iOS里面常常用到的工具类


## Podfile

To integrate DSUtil into your Xcode project using CocoaPods, specify it in your `Podfile`:

```ruby
source 'https://github.com/walkdianzi/DSSpecs.git'
platform :ios, '7.0'

target :targetName do

pod 'DSUtil', '~> 0.1.1'

end
```

Then, run the following command:

```bash
$ pod install
```
